## General Info
----------------
Generates a random message with topic of "Computer Notifications" using basic javascript and node.js.

## Technologies
----------------

Project is created with:

+ Node.js v16.2.0.

## Setup
1. clone :
    
    ```
    $git clone https://github.com/jr7072/r4nd0m-.git
    ```

2. run locally:
    
    ```
    $node r4nd0m.js
    ```
